Justifizer

Justifizer is a simple, lightweight, row based gallery layout utility with no dependencies.

With this html

<div class="gallery">
  <div class="gallery__item">
    <img src="http://lorempixel.com/250/300/"/>
  </div>
  <div class="gallery__item">
    <img src="http://lorempixel.com/300/300/"/>
  </div>
  <div class="gallery__item">
    <img src="http://lorempixel.com/350/300/"/>
  </div>
</div>

and this javascript

var fixGallery = justifize.bind(null, '.gallery');
window.onload = fixGallery;
window.onresize = fixGallery;

you get a gallery looking something like this.
